# BLTSEC

Source for [bltsec.com](https://bltsec.com) — offensive security research and operations.

## Stack

- [Hugo](https://gohugo.io/) static site generator
- [Terminal theme](https://github.com/panr/hugo-theme-terminal) with teal accent
- GitHub Pages + GitHub Actions for deployment

## Local Development

```bash
hugo server -D
```

## Deployment

Push to `main` — GitHub Actions builds and deploys automatically.
